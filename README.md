Python File Handling Project

Description:
This Python project demonstrates file management using CRUD operations. Users can create, read, update, and delete files easily. The program also lists existing files, ensures safe file operations, and provides an interactive interface for efficient data management.

Features:

->Create new files and add data
->Read and display file contents
->Update existing files
->Delete files safely
->View all files in the directory

Tech Used:

=>Python 3
->File handling modules: open, read, write, os, pathlib
